# 규칙 문서 인덱스

이 프로젝트에서 작업하는 모든 AI와 개발자가 따라야 할 규칙을 정의합니다.

## 규칙 카테고리

| 디렉토리 | 설명 | 상태 |
|----------|------|------|
| [common/](common/README.md) | 모든 AI에 적용되는 공통 규칙 | 운영 중 |
| [ai/](ai/README.md) | AI별 특화 규칙 | 준비 중 |
| [infra/](infra/README.md) | 인프라 관련 규칙 | 준비 중 |

## AI를 위한 규칙 적용 순서

1. **`common/` 규칙을 최우선으로 적용한다.**
2. 특정 AI에 해당하는 `ai/` 규칙이 있으면 추가로 적용한다.
3. 인프라 관련 작업 시 `infra/` 규칙을 추가로 적용한다.
4. 하위 규칙이 상위 규칙과 충돌하면 **상위(common) 규칙을 따른다.**
