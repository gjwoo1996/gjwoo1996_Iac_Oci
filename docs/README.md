# 문서 인덱스

이 디렉토리는 `gjwoo1996_Iac_Oci` 프로젝트의 모든 문서를 관리합니다.
AI 및 개발자가 프로젝트의 규칙과 설계를 파악하는 진입점입니다.

## 디렉토리 구조

```
docs/
├── README.md                        # 이 파일 — 전체 문서 인덱스
├── rules/                           # AI 및 개발 규칙 ("해야 한다")
│   ├── common/                      # 모든 AI에 적용되는 공통 규칙
│   ├── ai/                          # AI별 특화 규칙 (추후 확장)
│   └── infra/                       # 인프라 관련 규칙 (추후 확장)
└── guides/                          # 설명·계획·설계 문서 ("이렇게 한다")
    └── setup/                       # 개발 환경 설정
        └── devcontainer-setup-plan.md
```

## AI를 위한 안내

이 프로젝트에서 작업하는 모든 AI는 반드시 `rules/` 문서를 먼저 읽고 규칙을 준수해야 합니다.
규칙 우선순위: `common` 규칙 → `ai` 특화 규칙 (해당 시) → `infra` 규칙 (해당 시)

**규칙 문서 바로가기:** [rules/README.md](rules/README.md)
